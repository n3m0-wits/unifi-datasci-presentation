## Questions that will need to be answered:

1. Using a cross-correlation matrix, please explain if there are intercorrelated variables in your dataset.

2. Generate appropriate output explaining the distributions of the underlying variables used in the dataset. Also explain various properties of your data elements.

3. Considering the problem presented by the author, define your model’s explanatory and response variables, and give your findings of what potential type of model fits you’ve considered.

4. Do an appropriate test to see which data elements are normally distributed or not

5. Were any transformations / variable reduction techniques required on any of the data elements or as
input to the model?

6. Fit an appropriate model and explain the following:
a. How you’ve approached your model and variable selection criteria
b. How good of a fit was the model
c. Your model production “formula”

7.  In your own words, how can we use a learning algorithm to identify “good” vs. “bad” quality red wine, particularly since techniques may have changed since 2009 when this was first examined
